# NexoCetEx Front-end
El Front-end desarrollado con Ionic, Vue y Capacitor para el proyecto NexoCetEx como parte de la materia implementacion de proyectos

## Tecnologias utilizadas
Ionic Framework
Vue
Capacitor
Firebase Cloud Messaging

# Instalacion
npm install

## Ejecucion
ionic serve desde cmd

## Generar APK
npm run build
npx cap sync android
npx cap open android

# Desde Android studio
Build / Generate APK
